## About

I'm a professional .NET software engineer (~5 years in) shipping production systems by day and tearing apart binaries, clients, and game protocols by night.

Things I gravitate toward:

- Long-lived, maintainable backend codebases.
- Game dev, emulation, and odd little simulation projects.
- Tooling, automation, and reverse engineering / ethical hacking.
- Writing about what I learn so future-me (and other people) can stop suffering.

Outside of tech: husband, dad, and general tinkerer.
